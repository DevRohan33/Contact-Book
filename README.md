# Contact Book Web App

A simple Contact Book web application built with Flask, HTML, CSS, and JavaScript. This app allows users to manage their contacts by adding, viewing, and deleting contact information. The user interface includes tabbed navigation for better organization and is designed to look like a mobile app, limiting the width and centering the content.

## Features

- **Contact List:** View all saved contacts with name, phone number, and email address.
- **Add Contact:** Easily add new contacts by providing name, phone, and email.
- **Delete Contact:** Remove contacts with a single click using a styled button.
- **Tabbed Interface:** Navigate between the contact list and add contact forms using tabbed buttons.
- **Mobile App-Like Design:** The layout mimics a mobile app, with centered content and limited width for a neat appearance.
- **Responsive Design:** The app is responsive and adjusts to different screen sizes.
